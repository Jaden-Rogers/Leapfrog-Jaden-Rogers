# Leapfrog-Jaden-Rogers

This is a game where the objective is to jump over your opponent as many times as possible.
This game does not have a win condition

Player One Controls:

Forward - W
Right - D
Left - A
Back - S
Jump - Space


Player Two Controls:

Forward - I
Right - L
Left - J
Back - K
Jump - P
